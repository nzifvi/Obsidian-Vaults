Hopfield Network = simplest neural network.
- Hopfield network = a fully connected, single layer auto associative neural network.
- Only one layer.
- Each neuron is connected to each other.


- Hopfield network is small enough to easily understand whilst being able to recognise a few patterns.

Per 4 neurons, there will be 16 connections.


2^n - n = no of required connections
- minus n as neurons will NOT be connected to themselves.
- Suppose some hopfield network has 4 neurons
- 2^4 - 4 = 16 - 4 = 12 (required connections)
