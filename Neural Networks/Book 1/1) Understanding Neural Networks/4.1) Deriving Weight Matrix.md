[[4) Recalling Patterns]]

---

A weight matrix must be derived to "train" a neural network to recognise a specific input pattern.

To create a weight matrix, follow below steps:

1) Create blank matrix of n^2 dimensions.
	n = number of neurons.

2) Using blank matrix, calculate the input pattern's contribution matrix.
	- Contribution matrix is added to the actual connection weight matrix.
	- As additional contribution matrices are added to connection weight matrix, connection weight is said to "learn" each of the new patterns.

	a. Calculate bipolar values for input pattern.
	- Bipolar means representing binary string with -1s and 1s instead of 0s and 1s.
	- 1s remains 1s
	- 0s become -1s
	- Done because 0 is not the inverse of 1, -1.
	
    b. Transpose and multiply matrix by itself.
   c. Set leading diagonal elements to all be 0.
	 - Done as neurons do not connect to themselves.


## Useful info

By training a neural network to recognise a binary input pattern, you also train it to recognise it's inverse.
- Flip all 1s to 0s.
- Flip all 0s to 1s.
- Inverse is produced.
