[[2) Pattern Recognition]]

Auto association = how neurons communicate that a pattern is recognised in something presented to the neural network.
- A neural network that supports auto association will pass a pattern directly from input to output layer.
	- Appears if no work has been done.

EXAMPLE:

1) Image of traffic is fed to input layer of neurons.
	- If neural network recognises the traffic light, the output layer will present the same image.
		- Does not matter if image is different, if a traffic light is recognised,
	- If input layer recognises what neural network was trained to recognise, the input and output layers will be the same.

If input pattern is successfully recognised, it causes an auto associative neural network to pass the input neurons to the output neurons.


## Failed Pattern Recognitions
- If the neural network fails to recognise an image, or pattern, passed to the neural network, the output will be some other pattern that is different/distorted from what was inputted.
	- Output can look similar to human eye. It does not matter.
	- A failed recognition often results in a distortion of the input pattern.
