[[1) Neural Network Structure]]

---
- Neurons grouped into layers.
	- Layers = groups of neurons that perform similar functions.
	- Three types of layers.
	1) Input layers.
	2) Layer(s) between input and output layers.
	3) Output layers.

## i) Input Layers

Input neurons receive input from user program.

## ii) Layer(s) between input and output layers.

Multiple layers between input and output layer. These are hidden and never directly interact with the program.
- Can be 0 or more layers.

## iii) Output layer

Output layer sends data back to the user program post-processing.

![[Pasted image 20250206193718.png]]

- Input and output layers not just interface.
	- Any neuron, in any layer, can impact processing.
	- Processing can occur at any layer of the neural network.

- Neural networks can have different amount of layers. 
	- Even the hidden layer is optional.
	- Input and Output layers are REQUIRED.
		- One layer can be made to act as both Input and Output layer (Hopfield Neural Network.)
