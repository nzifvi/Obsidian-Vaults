[[1) Neural Network Structure]]

## Neuron
- Neuron = basic building block of neural networks.
	- Neuron is a communication node that receives an input and produces an output.
	- Neurons receive input from other neurons OR user program.
	- Neurons send output to other neurons OR user program.

- Neuron producing an output is known as the neuron activating or firing.
	- Neurons activate when the sum of their inputs satisfies neuron's activation function.

![[Pasted image 20250206190939.png]]
w = weights between a neuron and the neuron it is connected to.
k = neuron no. that the neuron is connected to.
x = input to this neuron from each of the other neurons.
u = sum.
- Every input, from each neuron, must be multiplied by the weight of the connection it is inputted into the receiving neuron from.

Above sum given to a neuron's activation function. If sum is greater than a threshold value, neuron will fire.
- Activation function tells current neuron whether to fire or not based on if the value exceeds some level.
- These types of methods aka Threshold methods.
	- Many threshold methods exist.

### Hyperbolic Tangent, TANH, activation method:

- TANH method is one of many activation methods.
- TANH method is fed the sum of input and weights, u, and calculates the hyperbolic tangent of u.

### Neuron Connection Weights
- Neurons are connected together.
- The value of these connections, aka weights, are not equal. Each connection can be assigned a different weight.
	- The weights of connections are what allows neural networks to recognise patterns.
	- If the weights are adjusted, neural network will recognise different patterns.

- Adjusting weights = very important operation.
	- Neural networks trained to determine how the weights should be valued.
	- Training = process of adjusting individual weights between each neurons until a desired output is achieved or NEARLY achieved.
