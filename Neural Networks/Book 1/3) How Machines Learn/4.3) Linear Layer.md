[[4) Types of Neuron Layers]]

Linear layer is pretty much no layer at all
- Least commonly used of all layers.
- Linear layer does no modification of pattern before outputting it.

### Linear Function

y(u) = u

### Properties of the Linear Layer

- Linear layer useful when you need entire range of numbers to be output.
- Neurons usually considered as on or off.
	- Because TANH and Sigmoid establish upper and lower values that are approached, they can be used for boolean type operations.
- Linear layer is useful for presenting a range, not on and off operations.
