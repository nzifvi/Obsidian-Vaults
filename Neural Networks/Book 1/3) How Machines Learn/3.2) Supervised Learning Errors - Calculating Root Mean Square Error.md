[[3) Error Calculation]]

The Root Mean Square error is used to determine whether the neural network has trained enough.
- RMS error can be calculated after calcErrors has been called.
- RMS calculated by getError() method.

```java

public double getError(int len){
	double RMS_err = Math.sqrt(globalError / (len * outputCount));
	globalError = 0;
	return RMS_err;
}

```

Above code...
1) Calculates RMS_error...
- globalError is divided by the product of the length of a training set multiplied by the number of output neurons.
- The square root of the resulting quotient is taken, giving the RMS error.
1) Clears the globalError for the next error calculations...
