RULES THAT APPLY WHEN NAMING VARIABLES, CLASSES, OBJECTS, AND METHODS.

1) Choose names that are descriptive, describing the purpose of it, and make it clear what it does.

2) Make meaningful distinctions between other things that are named.

3) Use pronounceable names.

4) Use searchable names.

5) Replace magic numbers with named constants, allowing the value of a number to have meaning.

6) Avoid encodings.
