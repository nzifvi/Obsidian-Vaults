## What is Overloading?

Overloading is the act of providing multiple definitions of a method, each with different parameter datatypes, number of parameters, and or return types.

- Method overloading, aka static binding, or compile-time polymorphism.
- Several methods in a class have same name but different parameters.  
	- Must have same return type.
- Which of these methods that are actually called depend on the parameters passed. 
	- This is called dynamic dispatch as the method called is determined during runtime.
	- The compiler resolves which method, from a set of overloaded methods, to call based on the arguments passed to the method call.
```java
class B2{
    public String print(String a){
    return a;
    }

    public String print(int loopFor, String a){
    String toReturn;
    for(int i = 0; i < loopFor; i++){
        toReturn = toReturn + a;
        }
    return toReturn;
    }
}

public class Main{
    public static void(){
        B2 b2Obj = new B2();
        b2Obj.print("a");
        b2Obj.print(2, "a");
    }
}



```

Output:
1 -> a
2 -> aa
