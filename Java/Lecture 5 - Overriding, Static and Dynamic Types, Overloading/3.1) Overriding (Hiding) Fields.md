### Hiding Fields
-  Attributes of a child class cannot be overridden like methods, instead they are hidden.
	- When a subclass defines a field, with the same name as an attribute from the parent class, the sub-class declares a new field.
	- The parent class field is hidden, it cannot be accessed any longer.
	- Hidden fields make code unreadable, DO NOT USE THEM.
