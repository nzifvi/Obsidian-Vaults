## What is Overriding?

Overriding is the act of changing the behaviour, not method signature, of a method a class inherits from a superclass or implements from an interface.

#### OVERRIDING $\neq$ OVERLOADING, THEY ARE 2 DIFFERENT THINGS

- Overriding a method means to redefine a method that was inherited from another class.
	- The final keyword prevents methods, and classes, being overridden.
		- Static methods cannot override object methods as they are class methods.
	- An overridden method WILL have the same name, number, and type of parameters. The return type of an overridden method will also be the same.
	- What changes is the behaviour of the method: the code can be rewritten and made to perform different behaviours, or incorporate new class attributes part of a subclass.

```java
@Override //@Override applied above method signature to override
public void myFunc(){
    //Some method
}
```
