- You can call an overridden method, from the object which overrides said method, the normal way any method is called.
	- However, you can also call the parent's definition of the method too by using the super keyword.

```java

class C1{
    public void print(){
        System.out.println("hello");
    }
}

class B1 extends C1{
    @Override
    public void print(){
        System.out.println("bye");
        super.print();
    }

    

    
}

public class Main{
    public static void(String[] args){
        B1 bObj = new B1();
        bObj.print();
    }
}

```

- Can only access parent method, using super, inside of the child class.
