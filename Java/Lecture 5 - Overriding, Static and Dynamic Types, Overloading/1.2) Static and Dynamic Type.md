### Static Type:
- The declared data type of some variable, or object, is it's static type.
	- Because it is declared in source code, it is part of the static (unchanging, fixed) representation of the program.

- Due to dynamic behaviour, the actual behaviour of a program can change from the static representation.

### Dynamic Type:
- The datatype currently stored in a variable is the dynamic type.
	- Dynamic type of a variable depends on assignments during runtime.

```java

public class Class1{

}

class Class2 extends Class1 {

}

Class1 cObj = new Class1(); //Static type of cObj = Class1.
cObj = new Class2(); //Dynamic type, at this current line, is now Class2.

```

- The dynamic type might ONLY be known during runtime. During compilation, only the static type is known.
	- During compilation, static type is used.
	- During runtime, dynamic type is used.
