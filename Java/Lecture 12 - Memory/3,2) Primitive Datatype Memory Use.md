## Primitive Values
Primitive values (int, boolean, char) are just the value and no extra data.

The amount of memory used is the same regardless of if a 32 or 64 bit machine is used.

#### Boolean and bytes are 1 byte.

#### int and float are 4 bytes.

#### short and char are 2 bytes

#### long and double are 8 bytes
