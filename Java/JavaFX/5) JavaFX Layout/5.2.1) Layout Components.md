[[5.1.1) JavaFX Layouts]]

---

Layouts are good for laying out [[4.1.1) JavaFX Controls]] 
### 1) HBox
- Adds components in horizontal box, fixing the height as the height of the largest component.

### 2) VBox
- Adds component into a vertical box: setting width to be the widest of it's components.


Components of both HBox and VBox are sized if box is shrunk.
