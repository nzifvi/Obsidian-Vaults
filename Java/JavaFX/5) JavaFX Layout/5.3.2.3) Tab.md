## What is a Tab

A Tab is essentially a sub scene. It can have it's own layout, such as BorderPane, which allows for multiple Tabs, in a TabPane, to have a different appearance.
- NOTE: A Tab can only have a single layout and this is set to the Tab using .setContent() method

```java
Tab entryTab = new Tab("Entry"); //Naming the Tab
entryTab.setCloseable(false); //Preventing Tab from being closed

//Now, create a Layout for the Tab (BorderPane is pretty good)
BorderPane bp = new BorderPane();
//Items can be attached to the BorderPane. The BorderPane, in this case, is what is set as the content of the Tab to be displayed on the Scene if that Tab is opened

//... some Layout created and added as bp's child

entryTab.setContent(tabContainer);

```

Ensure that the Tab is added as a child of a TabPane and then that TabPane is added to the Scene: enabling the Tab to be loaded.
