## What is BorderPane

BorderPane is a layout that separates the layout container into different regions that can have things loaded into it.

![[Pasted image 20250306223856.png]]

Using set methods, some components can be added to the above regions
	.setTop(Node e), setRight(Node e), setBottom(Node e), setLeft(Node e), and setCenter(Node e)
- NOTE: Panes can be stored within regions, allowing components to be structured.

Can ALSO get what is stored in the region of a BorderPane, using getX() (X = desired region). A Node value is returned.
