TabPane provides a way of having multiple Tabs, each with their own components, in a single Scene.

![[Pasted image 20250306222507.png]]

The TabPane is attached to the Scene object once all the set up is complete.

![[Pasted image 20250306222606.png]]

#### setTabClosingPolicy()

- This method controls whether tabs can or cannot be closed, based on a flag that is passed to it.
#### getTabs()

- Returns an array of Tabs, allowing them to be accessed or have a new Tab added to the array.
	- Added Tab objects become children of the TabPane object.
#### add()

- Add allows a tab to be added to the TabPane.
- The number of Tab objects added = number of tabs.

## Uses of TabPane

- EACH Tab in a TabPane can have it's own layout.
