### 1) FlowPane
- Adds components in either horizontal or vertical orientation.
- Components are resized if HBox or VBox shrunk.

### 2) BorderPane
- Lays out children in top, left, right, bottom, and center positions.
- Top and Bottom children keep to preferred heights and extend width.
- Left and Right keep preferred widths and extend height.
- Center extends height and width.

### 3) StackPane
- StackPane lays children down in order of the z plane: one on top of each other.
- Provides sample mechanism to build overlaid controls on images, or build complex shapes.

### 4) TilePane
- TilePane sets either preferred no. of columns or rows: depending on orientation.
- Added components are laid out in equal areas: calculated by largest component in terms of height and width

### 5) GridPane
- Allows for full flexibility in setting the preferred dimensions and aligment of EACH square in a grid: merging squares if required.
- Excellent for use in GUI builders (SceneBuilder) but not so much fun programming

### 6) TabPane

- TabPane provides way to switching views through Tabs.
- Each Tab has user friendly label and has a Node as it's content.
- First, setContent() on the tabs.
- Secondly, add Tabs() to the TabPane object.
