[[4.1.1) JavaFX Controls]]

---

- When > 1 component is used in a Scene, they need to be arranged.
	- JavaFX provides code that allows for common arrangements that automatically handle arrangements of components in Scene.

- Different Pane classes have different Layout algorithms.
	- Panes have set of children that are components.
	- A layout = a way of organising components when the region is resized OR new components are added.
	- Classic layout examples = rows, columns, stacks, tiles, etc.
	- Parent has an ObservableList of children.
		- Can access using getChildren() and, using add(), can add another child to the Parent.
