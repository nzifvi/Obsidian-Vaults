1) JavaFX application main class extends javafx.application.Application class
	- Control of program handed to Application class.
	- Application class used to initialise all dependencies for GUI: returning a Stage object to a method called start() once all dependencies initialised.

```java
public class Main extends Application{
    public static void main(String[] args){
        launch(args);
    }
    
	@Override
    public static void start(Stage pStage){
        Label txt = new Label("hello");
        FlowPane root = new FlowPane();
        root.getChildren().add(txt);
        Scene scene = new Scene(root);
        pStage.setTitle("Program Window");
        pStage.setScene(scene);
        pStage.show();
    }
}
```
