#### Events

When something happens, an event is raised.
- Event described by an Object.

JavaFX input components can raise events when the user does something to them.
- mouse click causes MouseEvent.
- button click, or menu item selected, causes ActionEvent
- frame closed, or minimised, causes a WindowEvent

Many more types of events, with different causes, exist.


#### Event Handling
Events, to have a purpose or action performed due to an event occurring, must have something that happens (the resulting action).

Event handlers = Objects, attached to components, and handle events that are raised.

To be a handler, the Object has to implement the relevant EventHandler interface.
- E.g. ActionEvents handled by EventHandler</ActionEvent/>
- Handler uses generic type Event so it can work for all types of Events.
