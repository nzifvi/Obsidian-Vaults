---

A SINGLE Object can do a lot, if not all, of the event handling for multiple components.
- However, one handle() method would have to do multiple different comparisons, actions, and etc to sort out which action to perform (not good.)
- Multiple handlers can exist for a single event.
- Best to use multiple inner classes that EACH handle a type of event.
	- Outer class handles a specific type of event.
	- Each inner class handles the event in a different way.
	- Above is possible implementation, others can be used.
