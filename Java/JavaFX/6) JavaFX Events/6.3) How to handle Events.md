---
In JavaFX, EVERY event has...
1) Target (the node on which an event occurred).
   A target can be window, scene, control, etc.
2) Source (where the event came from).
   Mouse, etc.
3) Type of event
   If source was a mouse, the event type could be mouse pressed.

JavaFX has an interface, called EventHandler, which is implemented with a class.
- The class is then called, if an event occurs, and calls a method called handle(ActionEvent event)
- Override handle(ActionEvent event) and, when an Event occurs, that method is called.
