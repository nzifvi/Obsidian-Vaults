[[4.1.1) JavaFX Controls]]

---

Scene = container for all the content.
- Within Scene, content is represented as a hierarchical tree aka Scene graph.
- Scene has a root node in Scene graph. This is typically the Parent node.
	- Parents can either be...
	1) A REGION: an area for Controls to have their layout managed and to have CSS applied. A Pane = subclass of Region.
	2) A GROUP: brings together child components, such as Graphics, to be managed together through transforms.
		- Groups have no layout.
		- Groups not focused on for this module.
