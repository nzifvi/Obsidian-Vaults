### 9) TextField
TextField provides an area for text to be inputted into the GUI via the keyboard.

```java
TextField inputField1 = new TextField();
```

- Can assign text to a TextField before user inputs data to explain what goes here but labels to a side of the TextField is better
	- Because, once first piece of data inputted, the string assigned vanishes.

```java
TextField inputField1 = new TextField("put a name here");
```

### 10) Label:
Label provides a way of adding labels, with strings assigned to them, to the program to explain various parts of something.
- Example could be what is meant to go into a TextField.

```java
Label inputLabel1 = new Label("First Name");
```
