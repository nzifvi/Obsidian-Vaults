## 1) MenuItem

MenuItem object is a selectable button in a Menu object.
- When initialising a MenuItem object, a string parameter can be passed which sets the name of the selectable option

```java
MenuItem save = new MenuItem("Save");
```

---

## 2) Menu

Menu is an object that has MenuItems assigned to it. Menu(s) are added to the MenuBar object.

![[Pasted image 20250306225808.png]]

```java
MenuItem save = new MenuItem("save");
MenuItem load = new MenuItem("load");
Menu fileMenu = new Menu();
fileMenu.getItems().addAll(save, load);
```

.getItems() returns the array storing all MenuItems for a menu.
- Can add new MenuItems to the array using a method from ObservableList called addAll().

.addAll(), in this case, allows multiple MenuItems to be added to Menu object.

---

## 3) MenuBar

MenuBar is a horizontal bar of Menu objects. Multiple Menu objects can be stored on the MenuBar.
- The MenuBar is what is added to the Scene, or a given Pane/Layout.

```java
//Creating a Menu...
MenuItem save = new MenuItem("save");
MenuItem load = new MenuItem("load");
Menu fileMenu = new Menu();
fileMenu.getItems().addAll(save, load);

//Creating a MenuBar...
MenuBar sceneMenuBar = new MenuBar();
sceneMenuBar.getMenus().addAll(fileMenu);
root.setTop(sceneMenuBar); //Imagine root is a BorderPane
```

.getMenus() returns an ObservableList (essentially an array) of Menus, allowing the Menus of the MenuBar to be interacted with.
- Has a method, called addAll(), which allows new Menus to be added.

Once done, add the MenuBar to scene root.
