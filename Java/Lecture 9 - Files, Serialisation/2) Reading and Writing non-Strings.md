FileReader class is reading from file (later discussed)

FileReader and FileWriter classes are character streams.

A FileOutputStream object works like a FileWriter object, except for allowing an array of bytes to be written to a file.
- Also useful for writing w/ non-default character encodings.

A FileInputStream allows for arrays of bytes to be read.
- ALSO can be used for raw data, such as images, etc.
