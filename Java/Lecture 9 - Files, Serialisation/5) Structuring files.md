## Structuring Files
Sometimes, arrays or data is stored in a file and has to be read a specific way. 

This can be done by having values, stored in a file, that when read cause a conditional to occur.

```
10 10
10 10
n
1 5
3 2
```

Suppose n indicates a new matrix is to be loaded...

Characters can be used to indicate something specific should be done with input.
