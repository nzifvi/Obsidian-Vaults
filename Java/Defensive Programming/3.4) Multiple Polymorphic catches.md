If some exceptions should be handled the same way, you can write a catch statement for them both instead of having to duplicate catch statements.

```java
try{
    //do some catch
}catch(FileNotFoundException e | SecurityException e){
    //Both Exceptions handled the same way
}catch(AnotherException){
    //A different Exception handled a different way.
}
```

- | operator is like logical OR but for Exceptions and try-catch statements.
