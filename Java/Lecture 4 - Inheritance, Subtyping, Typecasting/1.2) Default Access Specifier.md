Default aka package-private
- Default can ONLY be used with interface.
- Whatever has default access specifier is accessible by any other classes within that package, as long as it implements the interface.
- Classes within same package can access something with a default access specifier.
- Subclasses, unless in the same package, CANNOT access something with a default specifier.

```java
public class Main{
	public static void main(String[] args){
	
	}
}

interface Blueprint{
	default void print(){
		System.out.println("Hello");
	}
}
```

Default access specifier can be used to define a method's body to an abstract method (methods declared in an Interface).

CANNOT use a default abstract method in another file besides the one it is defined in.
- Use static, not default, access specifier for this.
