### What is Typecasting?
- Typecasting = process of converting a variable from one data type to another.
- Two types of typecasting exist:
	1) Widening (Implicit) casting.
	2) Narrowing (explicit) casting.
	3) [[4.2.1) Typecasting with Objects]]


### 1) Widening (Implicit) Casting:

- Widening typecasting automatically occurs when converting a smaller datatype into a larger datatype. No data loss occurs during process.
	- Widening typecasting automatically handled by Java.

```java
int myInt = 100;
double myDouble = myInt; //Widening casting occurs
```

- This is widening casting as double has larger range than int. Thus, Java handles it automatically.

#### Widening Order
byte -> short -> int -> long -> float -> double


### 2) Narrowing (Explicit) Casting
- Narrowing casting requires manual converting, due to the typecasting a larger datatype into a smaller.
	- Data loss may occur.

Widening Casting order: (smaller) byte -> short -> int -> long -> float -> double (larger)
Narrowing Casting order:  (smaller) byte <- short <- int <- long <- float <- double (larger)

```java
double myDub = 9.78;
int newInt = (int) myDouble; //Must manually type cast, stating the new datatype the variable is being cast to.
```

- When converting from double to int, the non integer part of the double is lost.
	- When typecasting a double, of value 9.78, to an int, 0.78 is cut off: result in newInt being 9.
