## Subtyping and Generic Types for Collections

Generic types and subtyping can be used in tandem to produce a collection of subtypes.

To do this...
1) Using a generic type, declare the type to be a supertype of the subtypes...

```java
//Suppose Car, Boat, Van, are subtypes of Vehicle type

ArrayList<Vehicle> vehicles = new ArrayList<>();

Vehicle v1 = new Car();
vehicles.add(v1);

v1 = new Boat();
vehicles.add(v1);

v1 = new Van();
vehicles.add(v1);
```

Each subclass object can be added to the collection, via subtyping.
- However, when accessing, if wanting to gain access to subclass specific methods, you need to type cast.
- To do this, MUST have a way to determine the type cast to apply based on a property or identifier.

Example:

```java
for(int i = 0; i < vehicles.size(); i++){
	if(vehicles.get(i).getClass().getName() == "Vehicle"){
	    Vehicle temp = (Vehicle) vehicles.get(i);
	    // Can access methods that only exist for parent class Vehicle.
	}else if(vehicles.get(i).getClass().getName() == "Car"){
		Car temp = (Car) vehicles.get(i);
		// Can access methods that only exist for subclass Car.
	}
	// ...
}
```
