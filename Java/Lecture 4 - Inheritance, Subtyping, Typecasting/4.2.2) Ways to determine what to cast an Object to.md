## Ways to determine what to cast an Object to
Sometimes, a set of objects may be various subtypes of a parent.
- Might want to access methods specific to those subtypes.
- Must typecast them to the class they are intended to be.
- Ways to do this exist, such as accessing the Class name the object is an object of.


```java
// Let array S be an ArrayList of objects that are subtypes of a class
for(int i = 0; i < S.size(); i++){
	if(S.get(i).getClass().getName() == "possibleClassName1"){
		possibleClassName1 temp = (possibleClassName1) S.get(i) // Type cast
		// Use whatever is required.
		// temp is an alias. Whatever changes are made to it are made to the same
		// object in the array.
	}
}
```
