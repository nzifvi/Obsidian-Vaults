# Fully Autonomy
Full autonomy is where a robot has complete control in it's operation.
- No remote operation by human supervisor.

Two approaches exist for full autonomy:
1) Heuristic Approach
2) Optimal Approach

## Heuristic Approach
The heuristic approach is implemented by hardcoded rules and behaviour.
- Not best at adapting to dynamic situations.
- Undefined behaviour occurs when encountering events not programmed for.

Heuristic approach is not guaranteed to be optimal.

Heuristic approach does not need complete environmental information.


## Optimal Approach
Optimal approach requires more environment information via the use of specialised sensors.

Planning is achieved through optimisation: finding optimal path. through environment, to accomplish task.

Optimal approach is achieved by...
1) Giving robot ability to model the environment.
2) Determine optimal solution via the environment model.
