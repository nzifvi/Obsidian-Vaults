# Notes
A critical issue in navigating an environment, for robotics, is continually learning about the environment whilst the robot remains running and independent of a controller.

Another critical issue in navigating an environment is the fact that features of the environment, such as sudden obstacles appearing, can occur. A robot has to be able to detect and adapt in the face of such an issue.

More traditional methods of robotic path finding make assumptions which can easily not be true in many situations.
- For example, a method called simultaneous localisation and mapping (SLAM), assumes that the trajectory of a robot, about a path, is decided by an external path planning policy.
- Issues with SLAM when the trajectory is dependent on a chosen path.
