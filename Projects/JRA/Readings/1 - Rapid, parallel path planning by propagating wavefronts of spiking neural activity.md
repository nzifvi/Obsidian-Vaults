# Reference
Ponulak, F., Hopfield J John. (2013) _Rapid, parallel path planning by propagating wavefronts of spiking neural activity_. Frontiers. Available at: https://www.frontiersin.org/journals/computational-neuroscience/articles/10.3389/fncom.2013.00098/full (Accessed 05/03/2026)

# Notes

Efficient path planning and navigation is critical for animals, robotics, logistics, and transportion.
- Propagating wavefronts by means of spiking neural activity, implemented by SNNs, provides a rapid model of path planning and navigation.
- Neural mechanisms of how humans and animals achieve spatial navigation and path planning are not entirely understood.

Presents a model where waves of spiking neural activity can both encode a given environment as well as path plan and navigate through the encoded environment.

Wavefront SNNs can be implemented in multiple, entirely unknown environments, to achieve successful dynamic environment navigation.
