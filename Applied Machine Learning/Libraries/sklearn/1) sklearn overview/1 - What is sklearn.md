# 1) What is sklearn
`sklearn` is a python machine learning library. `sklearn` can be used to...
1) prepare data for machine learning (clean, scale, encode, load into different formats)
2) train models using prepared data.
3) make predictions using trained models.
4) evaluate the performance of models.

`sklearn`can be used for the following problems...
1) classification
2) regression
3) clustering
4) dimensionality reduction
5) model selection
	- comparing models
	- validating models
	- tuning model hyperparameters
	- choosing models.
6) pre-processing
