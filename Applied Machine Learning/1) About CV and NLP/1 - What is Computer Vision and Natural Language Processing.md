# What is Computer Vision (CV)?
Computer vision is...
1) Using computers to understand image contents and video data acquired from real world.
2) Computing properties of 3D world from visual data in the form of measurements.
3) Using algorithms and representations of data to allow a machine to recognise...
	- Objects
	- People
	- Scenes
	- Activities 


# What is Natural Language Processing (NLP)?
NLP is..
1) Using computers to understand context of text data.
2) Computing properties of text data.
3) Using algorithms and representations to allow a machine to interpret meaning in natural language.
