# What is Dynamic Range?
The dynamic range is the ratio between the highest and lowest intensity values, for a given channel, that can be described. 
- Corresponds to a range of luminance (brightness) visible in an image.
- Interested in detail within middle range of luminance

By playing with more local contrast controls, an image can be made to appear to have a higher dynamic range than it actually does.
