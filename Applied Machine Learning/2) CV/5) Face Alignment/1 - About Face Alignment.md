# What is Face Alignment?
Face alignment is the process of locating a set of facial features, on a face, which describe the...
- Shape of a face.
- Pose of a face.

Face alignment normally facial locates features such as...
- Eye corners.
- Nose tips.
- Mouth corners.
- Jawline.
- Eyebrow points.

Face alignment sometimes referred to as facial feature detection or facial feature finding.

Face alignment is not just for humans.
- Anything with a face is valid.

# Uses of Face Alignment
Face alignment can be used for...
- Photo filters.
- Virtual try-on of lipstick and other products.
- Applying virtual avatars during real-time.
- Expression analysis.
