# Problems with Pixels
1) Pixels are a dense representation of a scene.
	- There are lots of pixels in a given image. A lot of the pixels in an image may not be useful when trying to do identify something in an image.

2) Pixels are not invariant to changes in.
	- Geometric transformations (translation, rotations, etc)
	- Lighting, small motion
