# Geometric Transformations
Geometric transformations of an image occur when an image is taken from different perspectives.
- Causes geometric variance.

![[Pasted image 20260323115821.png]]

Methods for feature detection must be able to detect the same feature in an image taken from multiple perspectives.

# Photometric Transformations
Photometric transformations occur when the intensity of pixels change due to differences in hue and lighting.
