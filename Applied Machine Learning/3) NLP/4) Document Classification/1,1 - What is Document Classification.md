# What is Document Classification
Document classification is a type of problem which involves being given a document and classifying it: determining what category, out of a set of possible categories, the document belongs to.

Some scenarios of document classification are:
- Given corpus of reviews, how many are positive.
- Given corpus of scientific articles, identify the ones discussing big cheese.

# General Problems in Document Classification

#### 1) Specifying what makes a document belong to one class versus another. 

#### 2) Adapting to Data Sources being dynamic.
- Specifications of classes may change over time.
- The sentiment of words may also change, resulting in different interpretations of documents.

#### 3) Unbalanced Classes
One class may have a lot more members than others.
