# What is NLP?
Natural languages are languages invented by humans to communicate with humans. Natural language processing (NLP) is the act of enabling computers to handle natural language inputs and outputs.

# Applications of NLP
1) Info retrieval.
	- find and rank documents online that are relevant to a query.
2) Question answering.
3) Machine translation.
	- translate documents from one language to another.
4) Text simplification.
5) Text summarisation.
6) Opinion monitoring.
7) User-content moderation.
8) Automatic recommendation.
9) Chatbots and virtual assistants.
10) MORE

# Why is NLP difficult?
1) Variation.
	- There are many different ways to express the same thing.
		- Different languages.
		- Different styles/genres.
2) Ambiguity.
	- Same bit of language can have many interpretations.
		- Much of understanding sentences depends on context
3) Creativity and Evolution.
	- Languages change over time.
		- New words, meanings, and interpretations can occur.
4) Sparsity.
	- So many different words, combinations, and possible contexts.
5) Grounding and World Knowledge.
