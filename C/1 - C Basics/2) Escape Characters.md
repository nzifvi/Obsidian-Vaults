## What are Escape Characters?
Escape characters, which there are many of, each having a different outcome, allows for characters to be specified that otherwise are impossible to.
- Example: newline character, \n

Every escape character has a backslash, \\, at the start. It is then followed by a specific character.
- Each character provides a different modification to the string literal.

## Types of Escape Sequence

#### \a
escape character a causes an alert in the form of beep or bell.

#### \b
escape character b causes a backspace.

#### \n
escape character n causes a newline to begin after the escape sequence in the string literal.

\\?
allows for a question mark in a string literal.

#### \r
escape character r allows for a carriage return to occur: going back to the start of the current line.
