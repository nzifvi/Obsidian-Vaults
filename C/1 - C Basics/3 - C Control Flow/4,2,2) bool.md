## bool
The type bool, from \<stdbool.h> library, allows true or false values to be stored.

```c
x = 2;
bool i = x == 2;
if (x == 2){

}

if(i){

}
```

Both if statements, found above, do the same thing.
- One uses a logical condition.
- One uses a bool variable.
