## What are String Literals?
String literals are a sequence of characters, enclosed within quotation marks (normally double quotations marks), that = a string value DIRECTLY in source code.
- Like a sentence, known when looking at the source code, in the source code.

```c
int main(void){
	char *string = "Hello";
	// automatically creates a character array of the required size (number of characters in string literal)

	char string2[3] = "123"
	//strings are character arrays.
}
```
