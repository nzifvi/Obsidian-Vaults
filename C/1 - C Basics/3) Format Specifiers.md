## What are Format Specifiers?
Format specifiers are a sequence of characters, usually starting with %.
- Each format specifier normally starts with %

Format specifiers usually used within functions like printf() and scanf() to DEFINE how data is formatted and interpreted when data is read from and written to a stream.

ESSENTIALLY: they are like placeholders for values that will be inserted into the output or input string.
- Each format specifier denotes a specific type, based on the characters used after the % symbol.
- The position of a format specifier is where a value will be put into the string literal.
## Types of Format Specifiers

#### %d
d for signed decimal integer

#### %i 
i for signed decimal integer

#### %f
f for float

#### %lf
lf for double

#### %c
c for char

#### %p
p for void* (a pointer address)
