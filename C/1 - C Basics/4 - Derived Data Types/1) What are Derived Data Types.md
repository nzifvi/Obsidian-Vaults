## What are Derived Data Types?
All data types that ARE NOT THE BASIC ONES are derived from the basic data types.

There are 4 forms of derived data types.

#### Type 1 and 2: Aggregate Data Types
Aggregate data types combine multiple instances of one, or several data types.

Arrays: Combine items all with the SAME base data type.
Structures: Combine items that may have different base data types.

#### Type 3: Pointers
Pointers are entities that refer to an object in memory.
