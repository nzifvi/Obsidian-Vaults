- Objects can be nested inside of objects. This is by creating a property which is an object.

```
const obj1 = {
	firstName: "unknown",
	lastName: "unknownB",
	obj2: {
		bankBalance: 1200,
		monthlyWithdrawal: 341;
	}
}
```

- Nested objects can be accessed by chaining the objects and then the property desired:
```
console.log(obj1.obj2.bankBalance);
```
