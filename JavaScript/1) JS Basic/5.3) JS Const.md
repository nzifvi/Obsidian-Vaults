- Const variables CANNOT be redeclared.
- Const variables CANNOT be reassigned.
- Const variables have Block Scope.

- Const variables must be assigned a value upon declaration.
	- They cannot be reassigned a value.

## 1) Constant Objects and Arrays:
- When declaring an object or array as a const, you CANNOT:
1) Reassign a constant value.
2) Reassign a constant array.
3) Reassign a constant object.

- YOU CAN:
1) Change the individual elements of a constant array, but not the datatype of the array.
2) Change the properties of a constant object.

## 2) Constant Arrays:
- You can change the elements of a constant array but not reassign the array.
```
const cars = ["Saab", "Volvo", "BMW"];
cars[0] = "Toyota"; //OKAY BECAUSE UPDATING ELEMENT.
cars.push("Audi") //OKAY BECAUSE ADDING A NEW ELEMENT.
```
- YOU CANNOT REASSIGN THE ARRAY:
```
const cars = ["Saab", "Volvo", "BMW"];
cars = ["Toyota", "Volvo", "Audi"]; //ERRORS BECAUSE REASSIGNING A CONSTANT ARRAY.
```

## 3) Constant Objects:
- You can change the properties of a constant object.
```
const car = {type:"Fiat", model:"500", colour:"White};

//Can update an existing properties
car.colour = "red";
//Can add a new property to an existing object.
car.owner = "Johnson";
```
- YOU CANNOT REASSIGN THE CONST OBJECT:
```
const car = {type:"Fiat", model:"500", colour:"White"};

//WILL ERROR BECAUSE REASSIGNING A CONST OBJECT.
car = {"type:"Volvo", model:"EX60", colour:"Pink"}; 
```
