[[3) JS Output]]
- JS has many methods. One of these is getElementById().
```
1. document.getElementById("demo").innerHTML = "Hello JavaScript";
```

- document.getElementById finds a HTML element, using the ID which is a parameter.
- The .innerHTML has two functions and which one of these functions occur is based on if the method is being used to get or set content.
	IF GET:
		If getting, .innerHTML will return the content within the element as a string: allowing existing content to be read.
	IF SET:
		If setting, the innerHTML will replace the content within the element that has the ID.

### GETTING:
```
let elementContent = document.getElementById("demo").innerHTML;
```

- string is assigned the returned string of the method.
### SETTING:
```
document.getElementById("demo").innerHTML = "hello123";
```

- html element, with the specified ID, is assigned the string value.

## NOTE:
- JS accepts both double and single quotation marks.
	"hello123" = 'hello123' | BOTH ARE ACCEPTABLE
