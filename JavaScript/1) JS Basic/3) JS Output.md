[[4.1) JS Statements]]
-  JS can output data in multiple ways.
1) Outputting into a HTML element, using innerHTML.

2) Outputting into the HTML output using document.write() method.
	- document.write() will overwrite the entire webpage with the outputting data. Only use for testing.

4) Outputting into an alert box using window.alert() method.

5) Outputting into the browser console using console.log() method.
