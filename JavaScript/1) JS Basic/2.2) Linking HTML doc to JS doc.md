- When linking a HTML doc to a JS doc that it should use, the <script></script> element is used.
	- The <script></script> element SHOULD be at the bottom of the <body></body> element.

	- Because loading scripts slows down the displaying of webpages.

	- NOTE: the script element can ALSO be placed in the head element.

- JS files will have the file type .js.

```
<html lang = "en">
	<head>
	</head>
	
	<body>
		<script src="myScript.js"></script>
	</body>
```

- The src property stands for script.
	- The value of script will be the file name and file type of the javascript file, joined together with a period.
