- Sometimes, many variants of the same object type must be created.
	- To create an object type, an OBJECT CONSTRUCTOR FUNCTION is used.
	- Good practice: upper-case first letter used for name of constructor.

```
function Person(first, last, age, eye){
	this.firstName = first;
	this.lastName = last;
	this.age = age;
	this.eyeColour = eye;
}
```

- The constructor can then be made to use multiple congruent objects.

```
const boss = new Person("John", "Doe", 50, "blue");
const wife = new Person("BallAche", "McGee", 14, "green");
```

- If a variable is assigned a constant value, not a variable,  then all objects made with that constructor will share that value.

- Constructors can also have methods.

## BUILT IN JS CONSTRUCTORS:

```
new Object() //constructs an object.
new Array() //creates an array object.
new Map() //creates a map object.
new Set() //creates a set object.
new Date() //creates a date object.
new RegExp() //creates a RegExp object.
new Function() //creates a function object.
```
