-  Variables can be declared in 4 ways:
1) Automatically.
2) Using var.
3) Using let.
4) Using const.
---------------------------------------
## 1) Automatically:
```
x = 5;
y = 6;
```

- Without using var, let, or const, and directly just assigning a variable a value, a variable can be declared and initialised.
## 2) Using var:

```
var x = 5;
var y = 6;
var z = x + y;
```
- var keyword used in all JS code from 1995-2015.
## 3) Using let:
- let and const keywords added to JS in 2015. Var keyword should ONLY be used in code for old browsers.

```
let x = 5;
let y = 6;
let z = x + y;
```

## 4) Using const:
- a const variable cannot be changed once declared and initialised. Use for variables which have a datatype that SHOULD NOT change (arrays, objects, vectors).

```
const price1 = 5;
const price2 = 6;
let total = price1 + price2;
```
