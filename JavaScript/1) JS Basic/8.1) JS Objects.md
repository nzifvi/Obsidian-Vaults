[[8.2.1) JS Object Properties]]

## Object Properties:

- Objects have properties. Properties are the variables of an object. 
- Properties can have values, just like a variable can.
```
car.name = "Fiat"; //Car object's name property is set to equal Fiat.
```
## Object Methods:

- An object can also have methods. Methods are functions which exist within an object.
```
car.start(); //Method of car, start, is called.
```

## JS Objects:
- Objects, in JS, are also variables. 
- Variables, besides objects, only contain one value.
- An object can contain multiple values that are assigned to objects.

### JS Object Definition:

- There are three ways to define a JS object:
1) Object Literal.
2) new keyword.
3) [[8.5) JS Object Constructors]]
#### 1) JS Object Literal:
- An object literal is a list of name:value (properties) encased within curly braces.
- object literals are ALSO called object initialisers.
```
const person = {firstName:"John", lastName:"Doe", age:50, eyeColour:"blue"};
```
- an object called person is created.
	- should be made const so the person object cannot be reassigned a datatype.
#### 2) new keyword:
- new keyword can be used to create an empty object. 
```
const person = new Object();
```
- properties can then be added to the object.
```
person.firstName = "john";
person.lastName = "doe";
```
