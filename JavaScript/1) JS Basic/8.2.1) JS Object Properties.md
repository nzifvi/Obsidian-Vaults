[[8.2.2) Nesting JS Objects]]

- An object is an UNORDERED set of properties.
- Properties are a vital part of objects.
- Properties can be...
1) Changed.
2) Added.
3) Deleted.
4) Some can ONLY be read.

## 1) Accessing JS properties:
- There are 3 ways, which all work, to access properties in objects.
```
// objectName.property
let age = person.age;
```
OR
```
//objectName["property"];
let age = person["age"];
```
OR
```
//objectName[expression]
let age = person[x];
```
## 2) Adding JS properties:
- Once an object exists, properties can still be added.
```
const person = [age:12]; //OBJECT CREATED.

person.nationality = "English" //NEW PROPERTY ADDED.
```

## 3) Deleting JS properties:
- Properties can be deleted from an object using the delete keyword.
```
const person = {
	firstName: "John",
	lastName: "Doe",
	age: 12;
}

delete person.age
```
