[[8.4) JS Object Display]]

- Methods are function definitions stored as a property within an object.

```
const person = {
firstName: "john",
lastName: "doe",
age: 12,
id: 123;
fullName: function() {
	return this.firstName + " " + this.lastName;
}
};
```

- this keyword is used to refer to the object that some statement is inside of.

- if the method isn't called with parentheses, it will return the function definition.

## Adding Methods:
- Methods can be added to existing objects.

```
const person = new Object();

person.firstName = "john";
person.lastName = "doe";
person.getName = function () {
	return (this.firstName + " " + this.lastName);
}
```
