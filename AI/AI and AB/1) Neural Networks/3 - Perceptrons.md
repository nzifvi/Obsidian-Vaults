# What are Perceptrons
Perceptrons are a single-layer, feed-forward network consisting of an input and output layer alone.
- Has no hidden layers.

![[Pasted image 20260128190704.png]]

Each output node is independent of other output nodes in the output layers.

# 2) What can Perceptrons Represent?
Perceptrons can be used to model simple Boolean functions and some complex ones.
- Perceptrons limited in how many Boolean functions that can be represented
