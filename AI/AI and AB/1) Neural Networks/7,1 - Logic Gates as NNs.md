## Logic Gates as NNs
Logic gates can be implemented by an NN.

Logic gates...
- AND
- NAND
- OR
- NOR
- NOT
... can be implemented by a single neuron.
- This is because these logic gates are LINEARLY SEPARABLE.


Whereas logic gates...
- XOR.
... require a hidden layer.
- This is because these logic gates ARE NOT LINEARLY SEPARABLE
