# What is Positive Feedback?
Positive feedback is a type of feedback loop which amplifies a state.
- Greater input, the greater the output.
- Output of stage t is input of stage t+1.


Positive feedback is...
- Destabilising: it amplifies a state.
- Divergent: it is an entropy process, rather than a process which approaches/maintains equilibrium.

In reality, positive feedback cycles do not continue forever.
- A factor, external to the feedback cycle, will limit it.

$$
\lim_{t\rightarrow \infty}\biggr[state(t)\biggr]=\infty
$$
# Positive Feedback as Amplifier
Positive feedback loops can be used to amplify some state over time.
$$
state(t+1)=state(t)+a
$$
