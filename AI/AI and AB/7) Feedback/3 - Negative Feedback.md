# What is Negative Feedback?
Negative feedback is a type of feedback loop which attenuates a state.
- State value decreases, approaching a equilibrium.

Negative feedback loops are...
- Stabilising: approach a value and maintain equilibrium about that value.
- Convergent: approach a value.

$$
state(t+1)  < state(t)
$$

In mathematical models of negative feedback loops, system settles in equilibrium.

$$
\lim_{t\rightarrow \infty}\biggr[state(t)\biggr]=x
$$
# Negative Feedback as Control
Negative feedback is used in control theory. Negative feedback can be used to stabilise and regulate a state based on some set of parameters.

Negative feedback is used in biology as a core of physiological function. Negative feedback loops are used to control...
- Body temp.
- Hunger.
