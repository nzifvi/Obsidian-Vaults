# What is Reinforcement Learning?
Reinforcement learning is a mechanism which allows agents to learn how to behave successfully to achieve a goal whilst interacting with an external environment.
- Agent learns with experience from interacting with environment.

Reinforcement learning, RL, works by...
1) RL shapes behaviour using reinforcement:
	- Agent takes ACTIONS in an environment.
	- Those actions change the state of the agent and give a reward to the agent.
	- Behaviour of agent improves over time as it learns

## Problems with Reinforcement Learning?
### 1) Robot may not know how the World works.
Robot may not know how the world works or the consequences.
- It has to learn these.

### 2) Rewards may be sparse, not continuous.
Rewards may be sparse instead of being continuous.
- Robot may have to do multiple actions before getting a reward.
