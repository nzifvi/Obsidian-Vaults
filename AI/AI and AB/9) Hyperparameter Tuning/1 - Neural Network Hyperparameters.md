# NN Hyperparameters

NN Hyperparameters are:
- Learning rate.
- Learning rate decay.uk
- Batch size.
- Number of hidden layers.
- Number of neurons per layer.
- Momentum.
- Epochs.
- Activation Function.

# Learning Rate
Learning rate, LR, determines the rate at which neural network adjust model parameters per epoch.
- Adjustments aka steps.

High LR = model adjusts more quickly.
- Higher risk of unstable performance and data drift.
Low LR = model requires more time to train and requires more data.
- Higher chance of achieving minimised loss.

Gradient descent optimisation, what trains neural networks, requires a LR.

# Batch size
Batch size is the set no. of samples a NN will compute before updating model parameters.
- Has effect on compute efficiency and accuracy of training.

Higher batch size = weaker performance.
- Adjusting LR, along with batch size, can increase performance.
