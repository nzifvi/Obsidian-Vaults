## Applying Bayes Theorem
Suppose we have a sample space, $S_1$, which represents the days it rains out of 12 days. A T represents it did rain and an F represents no rain.

$$
S = \{T, T, T, T, F, F, T, T, T, T, F, T\}
$$
Suppose we have another sample space, $S_2$, which represents if a car crash occurred on each of the 12 days. A T represents one did occur and an F represents that no car crash occurred.

$$
S = \{F, T, T, T, F, F, T, T, F, F, T, F\}
$$

With these 2 samples, two distributions can be generated to model the frequency probability of the two possible outcomes of the sample sets occurring.

Let random variable R denote the outcome if it is raining or not...
$$
R \in \{T, F\}
$$
Let random variable C denote the outcome of if a crash occurred or not...
$$
C \in \{T, F\}
$$
First, using data already collected, create probability frequency distributions of the two sample set's outcomes....

| R = r | P(R = r) |
| ----- | -------- |
| T     | 0.75     |
| F     | 0.25     |


| C = c | P(C = c) |
| ----- | -------- |
| T     | 0.5      |
| F     | 0.5      |
However, to do anything meaningful, a conditional probability distribution must be known.
