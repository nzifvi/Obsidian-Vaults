## Examples of Events having Set Theory Applied

Suppose we have a set of cars, S, of 10 cars.

Out of the set S, 4 events exist:
A = car is black
B = car is a manual
C = car uses petrol for fuel
D = car passed MOT

Using set theory operations, many sub events can be created...

$A\cup B$ is the event that a car, $c \mid c \in S$, is black or a manual.

$A \cap B$ is the event that a car, $c \mid c \in S$, is black AND a manual.

$(C^{C} \cap D)$ is the event that a car, $c \mid c \in S$, does NOT use petrol for fuel AND has passed an MOT.
