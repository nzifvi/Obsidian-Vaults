## Z-Scores

Z scores provide a measure of how far a specific value x is from the mean of the distribution, $\mu$, and is expressed using standard deviations.

$$
z=\frac{x-\mu}{\sigma}
$$
for some x.

- x is the value being analysed.


## What does a Z-Score tell?
The z-score of a value x tells you how many standard deviations the value x is away from the mean.

- if z = 0, the value x is the mean $\mu$
- If z = -a, the value x is a standard deviation(s) below the mean $\mu$.
- $if z = a$, the value x is a standard deviation(s) above the mean $\mu$.


### Interpreting z-scores
If z-score is positive, it means that the value x is above the mean.

If z-score is negative, it means that the value x is below the mean.

The greater the absolute value of z, the farther it is from the mean.


### Uses of z-scores

#### 1) Outlier Detection
Values with a large absolute value z score are outliers BECAUSE they are far away from the mean.
- As |z| -> $\infty$, x becomes more of an outlier.
- As |z| <- 0, x becomes less of an outlier.


#### 2) Standardisation
Because the z-score is divided by a normal distribution's standard deviation, this means z-scores are standardised.
- Any z-score, from any normal distribution, can be compared with another z-score from another normal distribution.

#### 3) Probability
The z-score of a value x is used in getting the probability of value x occurring.
- z-scores used in integrating the probability density function.
- z-scores used in lookup tables, aka z-tables, for area under the curve of bell curve.
