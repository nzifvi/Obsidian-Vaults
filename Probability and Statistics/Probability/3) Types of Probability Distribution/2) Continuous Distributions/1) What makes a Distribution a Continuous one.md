## What makes a Distribution a Continuous one?

For a distribution to be continuous, the random variable it models must have an infinite number of possible values within a given range.
- Cannot be described as a finite set, with a finite cardinality.


$$
S_{1} = \{ 1, 2, 3\}
$$

As $S_1$ has a cardinality not equal to infinity, and all of the possible values are known, a random variable from $S_1$ would be discrete.

$$
S_2 = [0, 1]
$$

However, $S_2$ defines a range with infinite possible outcomes.

0.1
0.01
0.001
0.0001
...
There are an infinite number of possible outcomes within this range, therefore, a random variable from $S_2$ would be continuous.
