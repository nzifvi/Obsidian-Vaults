## What makes a Probability Distribution a Discrete one
A probability distribution that is modelled on a random variable, $X$, is a discrete one IF $X$ has a countable number of possible outcomes
- The cardinality of $X$ is not infinity in other words.


Suppose $X \in \{1, 2, 3, 4, 5, 6\}$ 
- $X$ would be a discrete random variable.
- Consequently, a probability distribution based around $X$ would be a discrete probability distribution.

#### Summary
If a random variable is discrete, it can be used to create discrete probability distributions.

If a random variable is continuous, aka indiscrete, then it CANNOT be used to create discrete probability distributions.
