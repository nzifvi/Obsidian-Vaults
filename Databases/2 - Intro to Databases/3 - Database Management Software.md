## What is Database Management Software?
Database management software (DBMS) is software which has the ability to...
- Store data in an integrated, structured format.
- Enable users to retrieve and manipulate the stored data.


### Data Definition Language (DDL)
DDL is used to...
- Define the database.
- Specify the data types, structures, and constraints of future inputted data.

### Data Manipulation Language (DML)
- Insert data.
- Update data.
- Delete data.
- Retrieve data.
