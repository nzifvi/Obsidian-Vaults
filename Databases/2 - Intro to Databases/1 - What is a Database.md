## What is a Database?
A database is an integrated collection of data that is organised to meet the needs of one or more users.
- Data is structured in a logical way, based on relationships between data.

Databases model a real-world enterprise / structure / entity.
- Entity types (student, course, modules).
- Relationships (student x is enrolled in course y).
