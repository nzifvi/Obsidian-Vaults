## Database Concepts
![[Pasted image 20250524130909.png]]

A Relation is a table with rows and columns, it provides a structure.

Attribute Names is a name of a set of attributes. It is an identifier.

A Record is an entry in the collection, contains attribute values for that record.

An attribute value is a specific value of a given attribute for a record.
