This type of ERD referred to as Crow's Feet Notation.
## Parts of an ERD
### Simplified ERD entities
![[Pasted image 20250705193054.png]]

In an ERD, simplified entities are represented as a rectangle and the name of the entity type.
- Any attributes belonging to that entity type are abstracted.

### Relationship Types
Each relationship, between given entities, are denoted via a line. The type of relationship is denoted by symbols at either end of the line.

![[Pasted image 20250705193339.png]]
Name of relationships should be attached to relationships.

#### Many
![[Pasted image 20250705193522.png]]
Meaning: many of an entity attached to symbol.


#### 1
![[Pasted image 20250705193708.png]]
Meaning: one of an entity attached to symbol

#### Mandatory
Mandatory relationships denoted as a line above another symbol. Can have mandatory many, mandatory 1.

#### Optional
Mandatory relationships denoted as a circle above another symbol. Can have optional many, optional 1.
