## Entity Type
An entity type consists of multiple objects with the same properties (bikes have wheels, frames, etc, thus bike is an entity)

## Entity Occurrence
Each existing member of an entity type is referred to as an occurrence of that entity.

## Attribute
An attribute is a property that describes some part of the entity type.

## Relationship Type
An association between entities. Each type of relationship has a name which indicates it's meaning.
