## What is SQL?
Structured Query Language (SQL), like any database language, allows...
1) A database to be created and structured using relationships between entities.
2) Performing of data management tasks (addition, deletion, modification of data).
3) Perform simple and complex queries.

SQL comes in 2 parts:
1) Data Definition Language (DDL) [[1,2 - SQL DDL]]
2) Data Manipulation Language (DML) [[1,3 - SQL DML]]
