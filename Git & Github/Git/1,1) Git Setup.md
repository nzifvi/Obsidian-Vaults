## Git Setup
When setting up Git, it should be set up...
1) Giving it an email and name.

### Changing Git Username
```git
git config --global user.name
```
- prints the git username

Can be changed by providing an argument encased within single quotation marks

```git
git config --global user.name 'nameHere'
```

### Changing Git Email
```git
git config --global user.email
```
- prints the git email registered with the current git.

Can be changed by providing an argument encased within single quotation marks

```git
git config --global user.email 'emailHere'
```
