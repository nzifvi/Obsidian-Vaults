# Primer
Neuronal responses depend on many properties of a stimulus.
- Let $s$ denote an attribute of a stimulus which a neuron may be sensitive to.
- Let $\bar{r}$ denote the average firing rate.

Best way to represent the response of a neuron is to count number of spikes, fired during presentation of stimulus.

The average firing rate, written as a function of s, is called the neural response tuning curve (NRTC)
$$
\bar{r} = f(s)
$$
As the value of s changes, r changes too.
- Neurons produce a numerical, continuous response for all possible values of r.

The function f is the tuning curve.
- Describes a neuron's values of s which will make the neuron spike strongly, weakly, or not spike at all.
