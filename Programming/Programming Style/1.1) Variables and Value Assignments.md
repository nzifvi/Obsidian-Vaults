## Variable Rules
When creating a variable, it's name should give some kind of hint as to what it does.

There can be 2 types of variables and a variable's purpose usually determines what type it is.
1) [[1.2) Container Variable]]
2) [[1.3) Signal slash Flag Variable]]

## Value Assignments
When assigning a value to a variable, regardless of what type it is, it should be done as per the language requires.

However, when assigning values to variables, or even just declaring them, in a consecutive sequence, some structuring should be used to improve readability

```c++
int     maxSpeed;
int     maxVelocity;
bool    isEngineOn;
Engine  engineObj;
People* passangers;

```

Above is example of how to structure a block of variable declarations and or value assignments to variables.
