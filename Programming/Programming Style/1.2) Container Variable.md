## Container Variables
Container Variables have one purpose, store a value in them. They can be later used to be operated on or even used in logical comparisons. However, their ultimate use is to store a value.

As for any variable, the name should allow the use, and or meaning, of a variable to easily be understood.

## Naming Convention for Container Variables
Use camel case.

```c++
int maxSpeed;
```

Merge all words into one word. First word's letter is uncapitalised, all following word's first character are capitalised.

If the variable is a pointer, affix ptr, to mean pointer, at the front of the variable's name. This is to indicate a variable is a pointer and must be handled correctly.

```java
Engine* ptrEngineObj;
```
