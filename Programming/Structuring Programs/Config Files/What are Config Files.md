## What are Config Files?
Config files are a formatted file which programs/web applications/operating systems/etc use to...
- Customise/adjust settings of an application/system.
- Store parameters of a program.
- Store types of data.

### Why use Config Files over Generic Files
Config files should be used over generic files, such as a .txt file type, as config files provide a language instructing the reading program, if the language the program is in supports the given config file type, how to read it.

Config files are also industry standard and good practice
## Types of Config Files

##### 1) JavaScript Object Notation (JSON) [[1 - JSON Intro]]
JSON is an easy to read, simple config file type.
- Works with multiple languages (C++, Python, Java, etc).

JSON is commonly used for...
1) API communications between client devices and a server.
2) Config files: storing parameters for a program and allowing easy modification of a program from the config file vs having to scroll through a massive code base.
3) JSON can be used to store data in databases/files.
