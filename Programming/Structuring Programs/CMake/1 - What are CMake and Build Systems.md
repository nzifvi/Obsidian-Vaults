## What are CMake and Build Systems?
CMake is a build system for C/C++.

Build systems are a layer of automated processes, applied to a project/program, which handle the process of compiling source code into an executable OR even a library.

Build systems handle...
- Compilation.
- Linking libraries.
- Managing dependencies.
- Ensuring consistency when compiled with different compilers.

Build systems allow...
- Hard-coded paths to be avoided (directories).
- Building a package on more than 1 computer.
- Supporting of multiple compilers.
- Can compile and then execute a project WITHOUT an IDE.
- Program's structure can be defined logically.
- External libraries to be used.
