# Types of Uncertainty
Uncertainty can be...
- Expected or unexpected
- Irreducible or reducible

## Expected/Unexpected Uncertainty
Uncertainty is expected when a state is non-deterministic.
- Outcomes to a random variable with uniform probability to occur.

Uncertainty is unexpected when there is volatility.
- 

## Irreducible/Reducible Uncertainty
If uncertainty can be reduced by learning, the REDUCIBLE uncertainty.
- Aka epistemic (learning about volatility)

If uncertainty cannot be reduced by learning, it is IRREDUCIBLE uncertainty.
- Aka aleatoric.

# Sources of Uncertainty
An environment has many sources of uncertainty that agents have to learn about and behave based on.
- Environment has states which an agent can monitor.

An agent in an environment can use cues, from the environment, to learn about the environment.

Agents learn about sources of uncertainty over multiple iterations.
- Learn about expected reward/loss of a source of uncertainty.
- Learn about variability (is the reward deterministic)

Agents also learn how common or rare it is for something to be in a certain state.
- Aka state probability.

Agents also learn if the state's value is deterministic.
- If state has value A then B occurs.
