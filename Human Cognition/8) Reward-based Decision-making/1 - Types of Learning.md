# 1) Supervised Learning
Given data...
$$
(\vec{x_{n\times1}}, y_{actual})
$$
Where...
1) $\vec{x_{n\times1}}$ denotes a sequence of independent variables.
2) $y_{actual}$ denotes a dependent variable which is dependent on $\vec{x_{n\times1}}$

Supervised learning is a type of learning where the labels (what a set of inputs actual equals), is known.

The goal of supervised learning is to learn a function which maps $\vec{x_{n_1}}$ to $y_{actual}$ 

Some types of tasks which use supervised learning are..
1) Classification
2) Regression
# 2) Unsupervised Learning
Given data...
$$
\vec{x_{n\times1}}
$$
Where...
1) $\vec{x_{n\times1}}$ denotes a sequence of independent variables.

Unsupervised learning is a type of learning where the labels are NOT known.

The goal of unsupervised learning is to learn patterns from the data.

Some types of tasks which use unsupervised learning are...
1) Principle Component Analysis
2) Clustering

# 3) Reinforcement Learning
Reinforcement learning is a type of learning where an agent interacts with an environment.
- The environment provides numeric reward signals which the agent can learn from.
