## 1d)
One area that the study of cognition has inspired the design of AI systems is how the human primary visual cortex (PVC )has inspired the design of convolutional neural networks (CNNs). The PVC is a hierarchical system which represents an image as electrical signals that have been converted from light waves received by the eyes. Each hierarchy extracts increasingly complex features from the input. Furthermore, each hierarchy of the PVC has a receptive field which determines how much of the input image the layer focuses upon. As the hierarchy is traversed upwards, the receptive field increases.

CNNs, similarly to the PVC, are hierarchically organised into layers. The CNN, which also receives an input image, extracts increasingly complex features from the initial input image as the layer increases. In addition to this, another similarity is that the PVCs and CNNs are both used for the goal of object classification and perform at relatively similar levels. 

Differences still exist between CNNs and the PVC. Through experiments, humans have been demonstrated to have a bias towards recognising objects by their shape whilst CNNs have a bias towards recognising objects by their texture.

One of the neural mechanisms underlying attention is the dorsal attention network (DAN). The regions of the brain which are part of the DAN are shown to have correlated neural activity when measured by fMRI. The regions of the DAN are suggested to be associated with visuo-spatial attention and attention allocation. This has been demonstrated by a study on animals where neurons part of the DAN were inactivated: consequently decreasing the ability for the animals to successfully perform attention tasks.

## 2d)
Attention is a characteristic and property of multiple perceptual and cognitive mechanisms. The first characteristic of attention is that it has limited capacity. Everything cannot be paid attention to at the same time. Despite this, there is no hard limit of attention. The amount that can be paid attention to at any given time depends on context. Secondly, attention has a selection mechanism. The function of attention is to select information, such as external stimuli, that will be consequently prioritised. Thirdly, it is modulatory. Paying attention to an object does something to it. Lastly, it is subject to vigilance. Attention can be sustained over time.

One area in which attention artificial system is similar to human attention is exogeneous attention. Exogeneous attention is attention captured by an external stimuli in the environment. Saliency maps, inspired by early computer vision, provides a model to how exogeneous attention functions. Elements of a saliency map represent parts of an input image. The value of each element is how much attention it grabs. By modelling how attention grabbing a feature of an environment is, saliency maps model how attention grabbing an external stimuli is. Consequently, this models human exogeneous attention
