# Fusiform Face Area
Faces = special for humans.
- A higher visual region of the brain, specialised for processing faces, exists.

![[Pasted image 20260302100009.png]]

A CNN, trained to recognise both objects and faces, segregated itself to have 2 sub-systems:
- One object subsystem: specialised in detecting objects.
- One face subsystem: specialised in detecting faces.
- Perceptual functional specialisation occurred by itself, without need to explicitly training the model to perform these perceptual functions.

The human brain has a specialised face area, due to the fact that this is the most optimal way to recognise faces.
- When learning to classify faces & objects, perceptual functional specialisation occurs as a by-product.
