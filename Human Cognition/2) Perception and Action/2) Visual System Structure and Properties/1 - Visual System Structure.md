# Inputs to Visual System
Light waves are detected by photoreceptor cells in retina of eye.
- Forms basic inputs for visual system.
- Inputs are translated to electrical analogue signals: allowing brain to interpret them.

Input analogue signals processed by visual cortex.

![[Pasted image 20260224105844.png]]

Primary visual cortex has 2 parts: Left Visual Cortex (LVC) and Right Visual Cortex (RVC).
- Inputs from left eye processed by RVC, inputs from right eye processed by LVC.
