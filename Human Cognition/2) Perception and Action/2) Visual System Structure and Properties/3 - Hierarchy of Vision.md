# Vision Hierarchy
Primary Visual Cortex is hierarchically organised.
- Higher regions used to extract more abstract features from visual inputs.

![[Pasted image 20260224111710.png]]

Each region is used to extract different types of features from visual inputs.
- Higher regions extract more complex features from visual inputs.
- Each region is specialised in detecting a type of abstract feature from visual inputs.

The higher the region, the larger the receptive field.
- Let $recept$ be a function which numerically measures the size of a region's receptive field.
$$
recept(V2) > recept(V1)
$$
