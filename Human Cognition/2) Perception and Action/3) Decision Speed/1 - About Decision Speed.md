# What is Decision Speed?
Decision speed is a measure of how much "processing time" it takes for a decision to be made.
- Measuring decision speed done by collecting reaction times ($t_{reaction}$).

# What is Reaction Time?
As $t_{reaction} \rightarrow \infty$, the higher the complexity of the decision.
- Takes less time to determine to decide if you are hungry than deciding if you love somebody.

When reaction times change, you can determine if the way a task is achieved is different.
- If reaction time for doing a task is different than another task, the way that task is accomplished must also be different.

For example: visual search.
- Finding a symbol, a feature search, is unaffected by set size: reaction time is constant regardless of set size.
- However, conjunction search, find the x + x which is green, is a more complex task: as set size increases, reaction time increases.

# What impacts Response Time?
![[Pasted image 20260302102332.png]]

Response time is a function of...
- Decision time.
- Motor planning time.
	- How long to send signal to muscle to perform motion.
- Motor execution time.
	- Time it takes motion to be performed, after motor planning time has been achieved.
- And some random noise.
