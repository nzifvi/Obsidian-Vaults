# Choice Reaction Time Tasks
Choice reaction time task requires a choice to be made once stimulus appears.
- Is stimulus type A, B, C,... 

Time taken to complete choice reaction time tasks will include...
- Decision-making time.
- Response preparation.
- Time to perceive stimulus.
	- High intensity stimuli are responded to faster.

Choice reaction time tasks normally take $500ms \leq t_{reaction} \leq 2500ms$
## Example
Stimulus appearing: a bus appears.
Choice reaction time task: is the bus that has appeared a 3X or 25X.
# Simple Reaction Time Tasks
Simple reaction time tasks involve simply reacting to a stimulus appearing/occurring.
- Performing some task if a stimulus occurs.

Time taken to complete a simple reaction time task will depend on...
- Effector motor speed (index finger is faster than thumb).
- Time to perceive stimulus.
	- High intensity stimuli are responded to faster.

Simple reaction time tasks normally take about 150ms.
- Cannot respond faster than 100ms UNLESS task is something repetitive. 

# Hick-Hyman Law
Hick's Law:
>Reaction time increases logarithmically with number of choices (n).

$$
t_{reaction}=blog_{2}(n+1)
$$
- b is a constant
