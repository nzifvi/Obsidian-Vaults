# What is Forward Problem?
Forward problem is when you predict the effect (retinal image) from the cause (stimulus).

Forward problem is WELL-POSED:
> Multiple inputs map to one solution.

# What is Inverse Problem?
Inverse problem is when you predict the cause (stimulus) from the effect (retinal image).

Inverse problem is ILL-POSED:
> One input maps to multiple solutions.
