# Memory Categories
Memory is considered to have 3 overall categories:
1) Sensory memory.
2) Short-term memory.
3) Working memory
4) Long-term memory.

These 3 overall categories have subcategories:

Sensory memory has the following as sub categories:
1) Iconic Memory: visual
2) Echoic Memory: audio.
3) Haptic Memory: touch


# Short-term Memory

Most famous demonstration of short-term memory is digit span.
- Digit span commonly used in clinical and developmental tests to test short-term memory.

Simple version of digit span:
- Read a sequence of letters and or numbers one at a time (around one per second)
- Subject repeats back as many as they can remember in the same sequence.

People usually have a limit of 4-7 characters that can be remembered.
- Famous paper from Miller in 50s says people can remember 7 plus or minus 2.


## Chunking
Chunking is a method which is able to circumvent the apparent hard limit of 7 characters that can be stored in digit span.

Shown subsequences of the digit span sequence.
- Append the next subsequence onto the end of the current subsequence.

Suppose below sequence is a digit span sequence. Chunking works by...
$$
ABCXYZ123
$$
First show...
$$
ABC
$$
... as a sequence.

Then show...
$$
ABCXYZ
$$
... as a sequence.

Then show...
$$
ABCXYZ123
$$
... as a sequence.


When chunking, your short-term memory is limited by no. of chunks rather than no. of characters.

# Working Memory
Working memory is distinct from short term memory. 
- Working memory is a catch all for any mental process which uses memory to complete a task.

## Central Executive
Central executive is the manager of working memory.
- Does not store information itself.
- Controls and coordinates the other components of working memory.

The central executive is responsible for...
1) Switching between tasks.
2) Deciding what info to focus on.
## Phonological Loop
Phonological loop is a temporary storage system for sound-based information.

It has 2 components:
1) Acoustic Store.
2) Articulatory Loop

Phonological loop is temporal.
- Speech, or sounds, occur over time.

### Acoustic Store
Acoustic store, aka Phonological store, holds sounds and words in their acoustic form for very short time.
- Stores for about 2 seconds.

### Articulatory Loop
Articulatory loop refreshes the acoustic store by silently repeating information mentally.
- Inner voice is the articulatory loop.

Articulatory loop enables memory to be extended for verbal items by rehearsing them.

## Episodic Buffer
Episodic buffers acts as a multi-dimensional storage which integrates information from...
1) Phonological loop.
2) Visuo-spatial sketchpad.
3) Long-term memory.

Unifies information, from above 3 sources, into an episode.

Episodic buffers allows different types of info to be combined.
- Sound of a word, with the meaning of a word, and a visual image associated with the word.

## Visuo-Spatial Scratchpad
Visuo-spatial scratchpad temporarily stores and manipulates visual and spatial info.

Visuo-spatial scratchpad can store/manipulate...
- Shapes.
- Colours.
- Locations.
- Mental maps.

# Long-term Memory
Long term memory is memory which persists more than a few or ten seconds.

Several categories of long-term memory exists:
1) Declarative (explicit) memory.
	 - Episodic and semantic.

2) Non-declarative (implicit) memory.
	- Procedural/motor memory.

## Declarative (Explicit) Memory
Declarative memory is information which you can declare.

Declarative, aka explicit, memory has 2 sub-categories:
1) Episodic memory.
2) Semantic memory.

Episodic memory refers to information, or events, which happened in your life
- Knowledge of facts from events you experienced.
- Events which you have experienced.

Semantic memory refers to generic memory of facts.
- Knowledge that is not connected to any specific event. E.g. this is a chair, the world is round, I am a human.

## Non-declarative (Implicit) Memory
Non-declarative, aka implicit, memory is a type of long-term memory about things which cannot be precisely articulated.
- Procedures, which are sequence of actions/instructions, are non-declarative as they cannot be precisely worded individually. They must be described with a sequence of words.


Non-declarative memory often used to refer to actions: driving a car, dancing, etc.
