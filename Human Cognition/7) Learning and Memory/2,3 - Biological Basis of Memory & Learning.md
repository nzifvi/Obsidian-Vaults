# LTP Properties of Learning
LTP shows three important properties in relation to learning:
1) Cooperativity.
2) Associativity. 
3) Synapse specificity. 

The three properties describe how learning occurs in networks like the hippocampus.

## Cooperativity 
Cooperativity describes when weak inputs fed to neurons together produce an EPSP.
- Can cause LTP for all pathways.
- Whole set of neurons learns to change their behaviour.

## Associativity 
Associativity describes that a weakly stimulated pathway will not result in LTP by itself HOWEVER LTP can occur when weak pathway is concurrently stimulated with a strong pathway.

## Specificity 
Specificity describes that LTP will not occur in a pathway if that pathway is not stimulated.
- Even if pathways adjacent to pathway are strongly stimulated.
