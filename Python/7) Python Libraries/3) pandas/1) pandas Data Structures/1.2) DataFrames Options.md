## Max Rows
If a DataFrame has a large number of rows, pandas will only return the first 5 rows and the last 5 rows when printing a DataFrame.

The no. of rows returned when printing a DataFrame is defined by the `pd.options.display.max_rows` option.
- Assigning an integer value to this changes the max displayable rows for a DataFrame.

```python
pd.options.display.max_rows = newMax
```
