## Declaring a Python Class
A class is defined by...
```python
class className:
	# data and function members here
```

## Constructors
A constructor is defined via the ```__init__()``` function member (constructors in Python)

```python
class className:
	def __init__(var1, var2, var3):
		# do class initialisation
```

## Function Members
do it like C/C++ using def reserve word to declare and define a function member.

## `del` and `pass`
An object can be deleted via the `del` reserve word.

The pass statement is used to prevent a function, with no function definition, from erroring.
- All functions members of a class must have a definition.
