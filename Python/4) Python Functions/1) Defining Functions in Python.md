## Defining Functions in Python
As always, like in any language, a function
- Performs a specific task.
- Can accept parameters as input.
- Can return a result (although this is not mandatory).
- Can be called, aka invoked, multiple times.


To define a function, use the def keyword.
```python
def function_name(parameters):
	# function body
	return value # return is optional.
```
