## Accessing Tuple Items
Items of a Tuple can be accessed via their index

```python
aTuple = (0, 2, 4) #an x,z,y coordinate
print(aTuple[0])
```

```output
0
```

## Check if a value exists within a Tuple
```python
aTuple = (0, 1, 2, 5)
if 1 in aTuple:
	print("true")
```

```output
true
```
