## Python Comments

Python uses hashtag to create a single line comment

```python
# hello
```

Python does not really have multiline comments, just use a hashtag on each new line to continue the comment.

However, Python ignores string literals that are NOT assigned to a variable. You can use this to create a multiline comment without hashtags.

```python
"
This would be ignored by python as it is not assigned to a variable.
"
```
