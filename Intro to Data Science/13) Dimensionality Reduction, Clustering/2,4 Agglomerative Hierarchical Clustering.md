# What is Agglomerative Hierarchical Clustering (AHC)?
AHC is an agglomerative technique which builds up clusters by repeatedly merging closest pairs of clusters.
- AHC doesn't need a number of clusters in advance. It figures it out itself.
- Clusters can be hierarchical: a cluster can be within another cluster.
