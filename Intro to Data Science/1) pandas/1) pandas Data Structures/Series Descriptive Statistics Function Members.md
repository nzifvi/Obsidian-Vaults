---
Series objs have the following built in function members which provide a way to get descriptive statistical values about the data the Series stores...

- mean()
- median()
- mode()
- sum()
- min()
- max()
- std()
	- gets standard deviation.
- var()
- unique()
	- returns an array of all unique values in a Series
- nunique()
	- returns the number of unique values in a Series
- value_counts()
	- returns the frequency of each unique value in a Series
