## What is DataFrame?
A DataFrame is an object from pandas library. It is a matrix.
- Can label the columns.
- Takes 2 values per parameter on function call. Each parameter results in a row.
	- First parameter: column name.
	- Second parameter: list of values for column.

```python
d = {
	'col1Name': [...],
	'col2Name': [...]
}

df = pd.DataFrame(data = d)
```

As a matrix, matrix operations can be applied to the DataFrame...

## DataFrame Attributes

### T
T is a transpose operation which results in the matrix being transposed.

To transpose the DataFrame do:
```python
d = {
	'col1Name': [...],
	'col2Name': [...]
}

d.T # T operator transposes the matrix
```

### at
at is an operation allowing the element at a given index \[rowIndex, colAttribute\] to be accessed.
- rowIndex is a numeric value.
- colAttribute is the key assigned to the column that is desired to be accessed.

```python
df.at[rowIndex, colAttribute]
```
- use square brackets as opposed to regular function call.
- returns the element at the given rowIndex, colAttribute.


### columns
columns attribute prints a tuple containing a list of the column attributes and the dtype of the DataFrame.


### size
size attribute returns an integer value which is the number of elements in the DataFrame.

### axes
returns a list representing the axes of the DataFrame. It contains 2 individual lists.
- First list is the number of rows.
- Second list is a list containing the attributes of the columns.
