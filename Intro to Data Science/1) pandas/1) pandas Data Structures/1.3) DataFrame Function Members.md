# DataFrame Formatting Functions
## head()
`head()` function member returns the columns and a specified number of rows, starting from the top.

```python
import pandas as pd
df = pd.read_csv(filePath)

print(df.head(10))
```

## tail()
`tail()` function member returns the columns and specified number of rows, starting from the bottom.

```python
import pandas as pd
df = pd.read_csv(filePath)

print(df.tail(10))
```

- range.

## value_counts()
`value_counts()` is a function which returns a [[2.1) Series]] object containing how many times a value has appeared in a column for all columns.

## Column Specific Summary Functions
Each prior summary function can be applied to a specific column, as opposed to the entire DataFrame, by accessing the specific column by index.

```python
import pandas as pd
df = pd.read_csv(filePath) # read csv file into dataframe object

print(df["income"].sum())
print(df["income"].describe())
print(df["sex"].value_counts())
```
