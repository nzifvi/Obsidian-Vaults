# What are Decision Trees?
Decision trees are tree with multiple statements as nodes. Each statement leads to another based on the parent statement being true or false.
- Each node is a test of a single feature present within a data sample.
- Decision trees can handle other values than just true or false. They can also handle multiple values (Is favourite colour red, green, or blue).

Eventually, after a path of statements have been made, a decision tree makes a decision based on the path of statements that has been taken.
- Decisions are the leaf nodes of a decision tree.

Multiple types of decision trees exist, based on how things are classified...
1) Classification Trees.
	- When decision trees classify things into categories "is X, is not X, etc", it is a classification tree.
