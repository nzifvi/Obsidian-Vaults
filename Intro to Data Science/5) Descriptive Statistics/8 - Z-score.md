# What is a Z-Score?
A z-score, z, is a measure of how many standard deviations a point is from the expected value of a sample.

- Measure of how "rare" an outcome is, considering the expected value is the most likely value to appear in a Gaussian distributed sample.

A z-score encodes 2 pieces of information about a data point:
1) Distance from Expected Value:
	- The absolute value of a z-score denotes how far a data point is from the expected value.

2) Direction from Expected Value:
	- The sign of a z-score denotes which side of the expected value it is on.
		- If -z, z-score is on left of expected value.
		- If z, z-score is on right of expected value.

# Calculating Z-Score
A Z-Score can be calculated by...

$$
z = \frac{x-\mu}{\sigma}
$$
- x is the data point the z-score is trying to be calculated for.
- $\mu$ is the expected value of the sample: $E[X]$
- $\sigma$ is the standard deviation of the sample.
