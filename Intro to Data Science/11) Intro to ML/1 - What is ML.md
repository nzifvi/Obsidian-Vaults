ML = a subset of AI which utilises computer programs often referred to as models. These models...
- Are trained to achieve a task by...
	1) Establishing a training/learning process.
	2) Potentially learning from data.
	3) Without us assigning values to model's parameters.

- Without the need to provide explicit instructions for how the machine should learn.
	1) Including how to ensure a model can reason about inputs it hasn't seen before.

- Key factor in ML = getting computer to learn a generalisable mechanism to achieve a task.

# Why ML?
Some problems are very difficult or even impossible to solve with an explicitly defined solution.
- Identifying all possible versions of a single character and determining if a given character is that single character would be impossible. There are infinite possible formats it can take.
- ML can be used to recognise patterns in forming the single character and then determine if a given character is that one.



# Components of ML
Three main components of ML are...
1) Tasks:
	-  Problem(s) that require a mapping from data to desired outputs.
2) Features:
	- Characteristics of data that are used to describe domain objects.
3) Models:
	- Encode the required task mapping with some kind of mathematical model.

# Examples of ML Tasks

## 1) Classification
- Binary classification involves separating data into two distinct groups.
- E.g: separating people that are at risk from heart disease from those who are not at risk from heart disease.

## 2) Regression
- Involves mapping data to a real value of a function.
- E.g: giving a measurement of risk of heart disease based on some input data.

## 3) Clustering
- Separating data into different clusters based on their characteristics/properties.
- E.g: grouping people based on their genetic properties.

## 4) Collaborative Filtering
- Identifying rules or an association(s) from data.
- E.g: recommending treatments based on patient records. If x, y true then recommend z.

## 5) Dimensionality Reduction
- Better visualise data or pre-process it to ensure a model is better trained.

## 6) Reinforcement Learning
- Learning a policy to map the environment to behaviours in order to maximise a reward.
