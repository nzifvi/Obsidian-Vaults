## Declarations and Definitions
A declaration is a statement that introduces something (a variable, class, object, function) into a scope.

A declaration specifies a type of the named thing (a variable, function, etc).

Also specifies an initialiser (an initialiser value or a function body).

### Kinds of Declarations
There are many things that can be defined in C++, the most interesting are...
- Variables.
- Constants.
- Functions.
- Namespaces.
- Types.
- Templates.
