## Header Files
A header is a collection of declarations, defined in another file called a header (aka header file).

Headers are included into a file using  \#include, allowing whatever is declared within them to be used.

Headers are used to separate declarations into multiple files, allowing independent code to be separated: preventing massive single file programs existing.
- Improves readability.
- Improves maintainability.
