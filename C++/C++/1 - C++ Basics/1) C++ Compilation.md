## C++ Compilation
C++ is a compiled language.
- To get a C++ program to execute, it must first be translated from C++ source code into a machine readable equivalent of the program, aka object code.

The translation is done by a compiler.

![[Pasted image 20250517152210.png]]

### Types of Code

The code written by a programmer is known as source code.

What the computer executes is called an executable, which is aka object code, and or machine code.

C++ source code files have .cpp file extension

Object code files have .obj file extension (on windows).
- Unix has .o file extension for object code files.

### What the Compiler Does
The compiler reads the source code and attempts to make sense of the source code.

It checks to see if the program's syntax is correct.
- If incorrect, a syntax error occurs and compilation is aborted there and then.

Errors found by the compiler are referred to as compile-time errors.

If no syntax errors are spotted without any actual execution occurring, the compiler produces the object code file.
