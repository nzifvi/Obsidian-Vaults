## C++ Linker 1/2
Programs usually consist of multiple parts. These separate parts, aka translation units, must be compiled and the resulting object code files MUST be linked together to form an executable program.

The program that links all of the program's parts together is called the LINKER.

### C++ Libraries
Libraries are typically some code, that contains functions, structs, or many other type of things, that want to be accessed in another source code file.

Access is achieved using the \#include


## C++ Linker 2/2
Errors found by the linker are referred to as link-time errors.
