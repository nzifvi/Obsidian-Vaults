## Pre-defined Macro Definitions
C++ provides some pre-defined macro definitions that can be used throughout source code and have a replacement substituted in-place.

\_\_LINE\_\_ : an integer value representing the current line in the source code file being compiled

\_\_FILE\_\_ : a string literal containing the presumed name of the source file being compiled

\_\_DATE\_\_ : a string literal in the form "mmmm dd yyyy" which represents the date the source code began compilation.

\_\TIME\_\_ : a string literal in the form hh:mm:ss which represents the time the source code began compilation.
