## What is OpenGL?
OpenGL is an API which provides functions and objects which enable graphics and images to be manipulated in various ways.

OpenGL is a specification: specifies the output of each function and how the function should behave.
- Doesn't provide implementation, that is done by developers implementing specification.

Developers of OpenGL libraries are usually done by employees of GPU manufacturers.
- Each GPU supports specific versions of OpenGL, often specifically designed for that GPU series.
